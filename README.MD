# Telegram AI Chatbot (Local LLM)

Telegram chatbot powered by Node.js and local AI (Ollama + Llama3).
Bot can reply automatically with friendly persona.

## Features
- Telegram Bot integration
- Local AI (Ollama Llama3)
- Friendly persona
- Emoji support
- /reset command

## Tech Stack
- Node.js
- node-telegram-bot-api
- Axios
- Ollama (Llama3)

## Installation

1. Clone repository
2. Install dependencies
   npm install
3. Create .env file
   TELEGRAM_TOKEN=your_token_here
4. Run Ollama
   ollama run llama3
5. Run bot
   node index.js
